# hubot-wit

A script to use Wit.ai with Hubot.

## Installation

In hubot project repo, run:

`npm install hubot-wit --save`

Then add **hubot-wit** to your `external-scripts.json`:

```json
[
  "hubot-wit"
]
```

## Sample Interaction

```
user1>> hubot hi
hubot>> hi!
```
